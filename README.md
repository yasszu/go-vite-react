# Single Page Application (Go + Vite + React.js)

## Getting Started

Run server:

```shell
make run
```

Run linter:

```shell
make lint
```

Run prettier:

```shell
make pretiier
```

## Libraries
- [chi](https://github.com/go-chi/chi)
- [vite](https://github.com/vitejs/vite)
- [react](https://github.com/facebook/react)
